## Getting Started

### Prerequisites

- Node.js (>= 14.x)  
- npm or yarn  
- A running instance of your database  
- (Optional) .env file with proper credentials  

### Installation

1. Clone the repo  
   ```bash
   git clone https://github.com/rajat-sharma-3745/ShopSphere.git
   cd ShopSphere
   cd frontend
   npm install  
   cd ../backend
   npm install
  ```
